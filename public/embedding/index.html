About the Demo Applications (and Code!)

The demo applications access LLM and semantic (vector) search results via the Unacog API.

  1. Unacog API: session management through Unacog involves billing, configuration, and secure API authentication with revocable keys.
  2. Vector Query Endpoint: `unacog.com/sessionapi/vectorquery` enables querying of similar vector results for any prompt, returning top K results based on cosine similarity scores.
  3. Message Endpoint: `unacog.com/sessionapi/message` is used for sending prompts to the LLM, supporting both direct submission and pre-processed embedding.
  4. GitHub Repo: explore the demos at https://github.com/HuskerSam/unacogdemos for code examples that are easy to understand and implement.

Populate Vector Databases

  • A GUI fo upserting data to Pinecone
  • Scrape data or import it from CSV or JSON files
  • Metadata (like 'title', 'url', 'bookIndex', etc.) are available in document prompt template as handlebars fields, providing more context and advanced search features
  • Generate text lookup database, useful for small to big pipelines or text was not included in metadata

Serverless and Chunking

  • Pod Server or Serverless Configuration supported
  • Chunks are sized by sentence or token count
  • Chunk overlapping options are available to retain context
  • Option to include source text in metadata

LLM Prompts that embed semantic results

  • Create sessions that are connected to your Pinecone Index
  • Use prompt templates to embed data and guide LLM response
  • See semantic result details such as similarity score, metadata, and source text for each result.
  • Manage the LLM context limit by setting an embedding limit to determine the number of semantic results embedded in the prompt

• Import templates for quick document creation
  • Optional title field - if left blank, message will be used as title
  • Owner-only labels to categorize the document
  • Reference note field for relevant information. Only visible and editable by the owner
  • Set usage cap to limit the number of views per shared document. Viewable to anyone with document access, but only editable by the owner.
  • After creation, document link can be shared with other users to allow editing

The dashboard displays all the documents you have created or shared, and you can easily filter and find them by assigning labels to each document. Within the document, you can access the Options tab, which allows you to:

  • Add or change labels and reference notes for the document in the Owners tab. You can also use the clipboard button to share the document with others, but please note that there are no permissions, so use caution. The Owners tab also allows you to add reference notes visible only to the owner.

  • Modify the document details, such as the title or token usage cap, under the Details tab. The archive mode option in the Details tab stops all prompts in the document, preventing you from sending, selecting, rerunning or deleting prompts. By default, the document is not capped.

  • Import prompt templates. You can import prompt templates to quickly add prompts to your document without having to create them from scratch.

By utilizing these sharing and importing features, users can collaborate on creating efficient and effective templates that meet their specific needs. The clone feature and selected flag filter also make it easy to quickly create new documents that mirror existing ones.

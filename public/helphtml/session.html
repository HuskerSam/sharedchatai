Sessions Help

Copying and Pasting Sessions

Sessions can be copy and pasted to chat or email in both text and HTML format. Individual prompts and responses can also be copied to the clipboard.

Downloading Sessions as a File

Sessions can also be downloaded as a file and used to start new sessions. The gear icon on the dashboard or on the sidebar menu in the session shows the session options dialog.

Working on a Single Session in Real Time

Multiple users can work on a single session in real-time. To collaborate on a session, share the session link with other users. Any changes made by one user will be immediately visible to all other users in the session.

Cloning a Session

Any user in a session can clone a session to start a new session. The gear icon on the dashboard or on the sidebar menu in the session shows the session options dialog containing the clone button.

Introduction

Overview

PromptPlus Ai is a collaborative interface designed to offer superior control over your interaction with AI models like GPT-3.5 Turbo and beyond.

Our app not only allows multiple users or devices to work on a document in real time, but also provides unique control over your prompt history, backend parameters, and offers a series of tools designed to facilitate collaboration and streamline workflow.

Initiating a chat session

Can be done in one of two ways on PromptPlus AI:

  1. Create a New Session Document: each app screen has a create button (+ icon) that allows users to start a new session
  2. Join an Existing Session: Alternatively, users can join an existing chat session if they have access to a shared link. The shared link is generated by the session's owner and can be distributed to others for collaborative sessions or shared conversations. Upon clicking the shared link, users are directed to the ongoing chat session where they can participate in the conversation.

To view a document on PromptPlus, users must first sign in.

The following login options are available:

  1. Anonymous Login: This is recommended for first-time users whose only intention is to view documents. Please note that with anonymous login, you won't be able to revisit your profile, as this option doesn't create a permanent account.
  2. Email Login: This is a passwordless option. Upon choosing this method, we'll send a login link to the email address you provide. Use this link to access your account.
  3. Google Login: With this option, you can use your existing Google account credentials to log in. This is our recommended method, as it allows easy and secure access to your account.

Once you're on the chat interface page, you can start entering your prompts for the AI to respond.

Key Concepts

Understanding Large Language Models (LLMs) Work

Large Language Models (LLMs) like GPT-3 are trained to predict the next word or token in a sentence based on the context provided by the preceding words or tokens.

For instance, given the input "The sky is...", the model might predict "blue" as the next word. However, as the models employ a probabilistic approach, the same input can yield different outputs, making these models non-deterministic. Understanding the foundational mechanics of LLMs is crucial to optimizing their use and potential.



Understanding and Managing Token Limits

The token limit in Large Language Models (LLMs) like GPT-3.5 Turbo is 4096 tokens per conversation. This limit dictates the AI's context, meaning once reached, the earliest parts of the conversation are omitted from the AI's consideration.

In many AI interfaces, this isn't made explicit, potentially leading to confusion if the AI responses lack context from earlier messages. PromptPlus AI addresses this by keeping users informed about their token usage in a session, ensuring they are aware of the AI's current context.

See engine for details on each model's token limit

Promptplus will alert you to exlude previous messages when we detect you may go over the model's token API limit

Our interface allows users to manually select which prompts are included or excluded from the AI's context. This gives you control over the information the AI considers, optimizing your interaction within the token limit, and improving response quality. In short, understanding and managing token limits can significantly enhance your AI interaction.

Chat Session Interface

Overview

TThe chat session interface forms the heart of the PromptPlus AI experience. This is where you interact directly with the AI, entering prompts, receiving responses, and leveraging powerful features designed to optimize the AI's performance.

Designed for Collaboration

In a shared document/session, all users are able to interact with the AI model through a centralized chat interface. The messages sent by each participant are marked with their respective username, ensuring clarity in the dialogue flow. The features available to all users include:

  1. Sending Prompts: Every user in a PromptPlus AI session can send prompts to the AI, enabling a dynamic conversation. PromptPlus AI uses color indicators for tracking changes in engine parameters. If the 'max tokens' parameter changes, the token predicted amount above the send icon turns yellow . For alterations to temperature, top_p, frequency penalty, or presence penalty, the icon turns orange . This visual cue allows users to promptly recognize parameter modifications, ensuring effective and well-informed interactions with the AI.
  2. Include/Exclude Prompts: Each user can decide whether which prompts are included or excluded from the context that the AI considers when formulating its responses. This is controlled by a select box attached to each prompt. Deselecting prompts allows you to remove or bring back the content into context quickly. Any participant can choose which messages the AI should consider while formulating a response. This way, users have the ability to steer the conversation, decide the context for the AI's responses, and influence the narrative collectively.
  3. Rerun Prompts: Any user can decide to rerun any included prompts to get a new response from the AI. Rerun a prompt with the same or different parameters. This can be especially helpful if the initial response from the AI wasn't satisfactory or if the users want to see a different perspective on the same prompt. Rerun works by moving selected message to the front of the request queue and rerunning it based on your current engine parameters. For best results, select on the message you want in the context history. Rerun allows any user to resubmit included messages to the AI for generating a new response, however when you rerun another user's message, the message will become yours.
  4. Change Parameters: All users have the ability to adjust the AI's response parameters, like temperature and top_p. This can shape the nature of the AI's responses, whether you want them to be more creative, more factual, or anything in between.

While anyone may deselect or rerun a prompt, only message owners are able to delete:

  • Delete Prompts: delete a prompt from the session chat.

In addition to providing a platform for interaction with AI, the chat interface is equipped with stats and features that provide feedback on token usage, aiding in cost management and comprehension of the model's processing.

  • Token Visualizer: Positioned at the top of the text area, the token visualizer offers a graphical representation of the model's tokenization process, providing insights into how text is dissected into tokens. Tokens are integral to how GPT models operate and calculate cost - the cost per token varies on a per-model basis.
  • Completion Tokens: For each prompt or message, the number of completion tokens utilized is displayed at the top right of each message. Completion tokens are the number of tokens used in the AI's response to a particular prompt. This helps you understand how much of the model's capacity was used for generating each response.
  • Total Tokens: The total token count for each prompt, encompassing both the user's input and AI's response, is displayed at the top of each message, below the activity timer. This aids in tracking the total token usage and managing the context that the AI can keep for subsequent responses.
  • Activity Timer: Each message is accompanied by an activity timer that shows the time elapsed since the message was sent. This timer updates every time the page is loaded, giving a clear indication of the age of each message. For messages older than 24 hours, the time is delineated in a month (3 character abbreviation) and day format.

Chatgpt Parameters

Changes from default values are indicated with color cues.

Like the chat interface, changes in the engine parameters are reflected in the UI.Yellow denotes changes in max tokens, while orange represents modifications to other parameters.

PARAMETER DESCRIPTION VALUE NOTES
max_token The maximum number of tokens that the AI model is allowed to generate in response to a particular prompt model dependent affects the cost per API call and the time it takes to get a response
PARAMETER DESCRIPTION VALUE NOTES
temperature Governs the randomness of responses. Higher values create diverse but potentially less focused replies, while lower values yield more focused and predictable answers. 0 to 2 Can be thought of as creativity dial. Generally recommended altering this or top_p but not both.
top_p Controls the word range for the AI's response. Lower values make it stick to more common words, whereas higher values allow it to consider a broader vocabulary. 0 to 1 Generally recommended altering this or top_p but not both.
Frequency Penalty High values discourage the AI from repeating itself, whereas low values make the AI more likely to reiterate previous phrases or sentences -2.0 to 2.0 only apply when there is existing conversation history. They don't affect the AI's initial responses in a new conversation
Presence Penalty High values encourage the AI to bring up new topics in the conversation, while lower values make the AI stick to the existing topics. -2.0 to 2.0 only apply when there is existing conversation history. They don't affect the AI's initial responses in a new conversation

Note Promptplus AI simplies values into a slider bar. Values are for informative purposes only. For more information on these parameters, visit OpenAI's documentation.

Sessions Dashboard

Overview

The Sessions page is your hub to manage and access all documents/sessions created or shared with you. This section provides a comprehensive overview of all the sessions in a user-friendly file system.

Here are the key components of each line item, from left to right:

Share Icon

Click on this icon to get a shareable link for the session.

Icon color indicates whether the session is shared or not.
The color coding is as follows:

  • link indicates you are the owner.
  • link indicates that you are the owner and you have shared the document.
  • link indicates that the session is shared with you (you are not the owner).

Document Title

The title of the document, which you can set during creation or edit later.
If you don't provide a title during creation, the first prompt is used as the document name.

Prompt Counter

Shows the total number of prompts executed in the session.
This feature helps you identify the desired session quickly.

Last Activity Date

Shows the date and time of the most recent prompt request in the session

Labels

Labels can be assigned during creation or added later to help you filter and categorize your sessions. The default labels provided are "business", "personal", and "archived".

Default labels can be changed in your profile settings.

Users

Displays the users in your session and their current online status.

Document Options

The Document Options can be accessed from two locations: in the chat interface, it's located near the session title, and within the dashboard, it's available for each individual session. The Document Options enables users to exercise a high degree of control over their sessions. This panel is divided into three tabs: Prompts, Options, and Owner.

Aside from the prompt tab, most of document options are availble to users initally when they create the session. The Document Options includes several features:


Prompts Tab

This tab focuses on export options for your sessions. You can choose to export the entire chat history or select specific parts to download, providing flexibility in managing and preserving your session data.

You can export your chat history in the following formats Text, HTML, JSON, CSV.

Whether it's sharing insights, backing up work, or reusing a session, the export feature has got you covered. Export the entire chat history or choose specific parts to download.

NoteUse Text and HTML for sharing. CVS and JSON are best for sharing session data.


Options Tab

This is where the majority of the session control tools reside

  • Edit title: Allows you to modify the title of your session.
  • Edit system message: Modify the system messages for a personalized experience. A system message gives context to a session without invoking a response.
  • upload: Use this feature to upload and integrate previous prompts into your current session. Make sure to use JSON or CSV.
  • Delete Session: Remove a session permanently.
  • Archive Session: If you want to preserve a session without any further prompt requests, use this option to make the session read-only.
  • Clone Session: Create an identical copy of the current session. This is especially useful when you want to try different approaches without modifying the original session.
  • Token Usage Stats: Gives you a detailed report on token usage for the session, helping you to monitor and manage your resource usage more effectively.
  • Clone Packet: Click to get a JSON of the packet info sent to API

Owner Tab

As an owner, you have additional tools at your disposal:

  • Add Labels: This feature allows you to classify your sessions using custom labels, which aids in the organization and filtering of your documents.
  • Owner's Note: A space for you to write notes related to the session. These can be reminders, observations, or any information pertinent to the session.

Profile Setting

Overview

The Profile Options, accessible from both the chat interface and session dashboard, is an essential tool for personalizing and controlling your user experience in PromptPlus. The dialog presents the user's registered email at the top and the sign-out option at the bottom. It's structured into three tabs: User, Labels, and Account.


User Tab

Use this tab to personalize your profile by changing your name and profile image

  • Name: Use the pencil icon to manually change your name, or the dice icon to generate a random one.
  • Profile Image: Upload a custom image or choose to generate a random mascot image.

In addition to these, there are four checkboxes for customizing display settings:

  • Large Text: Check this box to increase the size of the chat interface text.
  • Monospace Font: Switch the font family to monospace for chat prompts and responses.
  • Prefix Name: Use your name as a prefix to enable the AI chat to recognize who is speaking. This is particularly useful when there are multiple participants in a chat. It is generally recommended to include a system message that lets the AI know there will be multiple participants in the chatroom.
  • Auto Exclude: This feature automatically excludes the oldest prompt in your chat session to avoid going over the API token limit for each ChatGPT model.

Labels Tab

This section presents default labels (Personal, Business, and Archived) that you can assign to sessions during creation or afterwards. You can modify these labels according to your needs for better organization and filtering of your documents.


Account Tab

This tab provides critical information about your usage and subscription.

  • Monthly OpenAI Token Usage: Displays the total token limit and your usage for the month.
  • Token Usage History: Provides a detailed breakdown of token usage by day, month, year, and all time based on sent messages and replies from the AI.
  • Subscription Level: Shows your current subscription level. During the beta phase, all users are identified as Pre Release.
  • New Email: Allows you to migrate your document data to another email account.

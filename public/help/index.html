Foundations link

Getting to Know Large Language Models (LLMs)

Ever wondered how a Large Language Model, like GPT-3, comes up with its responses? Well, let's pull back the curtain a bit.

Think of LLMs as super-smart word predictors. They use the words you've given them to guess what word should come next. For example, if you say, "The sky is...", it might guess "blue" because that's a common next word in sentences it's seen during training.

But here's the exciting part - it's not always "blue". These models use something called a probabilistic approach, which means the same question could have different answers. So, sometimes, the sky could be "clear" or even "full of stars" - it all depends on the context!

Understanding how these models work helps us get the best out of them, whether we're teaching a class, exploring a research question, or just having fun.

And for those of you who love diving into the nitty-gritty, understanding this foundational concept is key when it comes to fine-tuning how the AI responds - do you want it to be more creative or less repetitive? You'll get the hang of it!

So, whether you're a newbie exploring the world of AI or a seasoned expert, this concept will help you navigate your interactions with our platform effectively. Let's get started!

Using the Chat Interface link

The chat session interface forms the heart of the PromptPlus AI experience. This is where you interact directly with the AI, entering prompts, receiving responses, and leveraging powerful features designed to optimize the AI's performance.

Designed for Collaboration

In a shared session, PromptPlus AI enables all participants to interact with the AI model via a unified chat interface. Usernames distinctly label each contribution, maintaining clarity throughout the dialogue. Features available to all participants include:

Prompting

Every participant in a session can send prompts to the AI, fostering a vibrant conversation. Color-coded indicators monitor shifts in engine parameters. If the 'max tokens' parameter alters, the predicted token amount above the send icon turns yellow.

Adjustments to temperature, top_p, frequency penalty, or presence penalty shift the icon to orange. This immediate visual feedback helps users recognize parameter changes, enhancing effective and informed AI interactions.

Prompt Inclusion/Exclusion

Each user can determine which prompts to include or exclude from the AI's response context, controlled by a selection box attached to each prompt. Exclusion of prompts allows users to modify the context swiftly. Each participant can influence the AI's consideration when generating responses, granting collective control over the conversation and narrative.

Prompt Rerun

Users have the option to rerun any included prompts to obtain a fresh AI response. Rerun a prompt with the same or different parameters, particularly useful if the initial AI response was suboptimal or if alternate perspectives on the same prompt are desired. Rerun operates by placing the selected message at the front of the request queue and resubmitting it based on the current engine parameters. For optimum outcomes, include the desired message in the context history. When rerunning another user's message, the ownership of the message shifts to you.

Change Engine Settings

All users are equipped to modify the AI's response parameters, like temperature and top_p, customizing the AI responses to be more creative, more factual, or anywhere in between. Please refer to the parameters section below for details.

Delete Prompts

While anyone may deselect or rerun a prompt, only message owners are able to delete. Delete removes a prompt from the session chat. Currently we delete the document data so it will also not be retrievable by us.

Feedback on Token Usage

In addition to providing a platform for interaction with AI, the chat interface is equipped with stats and features that provide feedback on token usage, aiding in cost management and comprehension of the model's processing.

Token Visualizer

Positioned at the top of the text area, the token visualizer offers a graphical representation of the model's tokenization process, providing insights into how text is dissected into tokens. Tokens are integral to how GPT models operate and calculate cost - the cost per token varies on a per-model basis.

Total Tokens

The total token count for each prompt, encompassing both the user's input and AI's response, is displayed at the top of each message, below the activity timer. This aids in tracking the total token usage and managing the context that the AI can keep for subsequent responses.

Completion/Reponse Tokens

For each prompt or message, the number of completion tokens utilized is displayed at the top right of each message. Completion tokens are the number of tokens used in the AI's response to a particular prompt. This helps you understand how much of the model's capacity was used for generating each response.

Activity Timer

Each message is accompanied by an activity timer that shows the time elapsed since the message was sent. This timer updates every time the page is loaded, giving a clear indication of the age of each message. For messages older than 24 hours, the time is delineated in a month (3 character abbreviation) and day format.

Threshold Dialog

The Threshold Dialog is a valuable feature located within the Document Options. This handy tool provides crucial data about your session, particularly concerning maximum token usage. It's designed to offer you a clear and comprehensive view of your token count, so you have better control over your interaction with the AI.

  • Max Token
    This displays the parameter setting set in the Engine, which governs the maximum token count for an AI response.
  • New Prompt
    This shows the token count of the current input in the text area, which hasn't been submitted yet.
  • System Message
    This reveals the token allocation used in the system message.
  • Prompt Count
    This displays all prompts as a ratio between selected and unselected responses, giving you a snapshot of your prompt selections.
  • Request Size
    This reflects the current token count in the context history (i.e., selected prompts), plus the value set for max tokens.
  • Model Limit:
    This indicates the token limit for the selected model.

In addition, the Threshold Dialog provides an option to automatically exclude prompts that exceed the API limit, a function that is often performed invisibly in other app interfaces. Remember, understanding how tokens are used and managed in the interface is key to maximizing your interaction with the AI model.

Understanding and Managing Token Limits

Each Large Language Models (LLMs) has a token limit. GPT-3.5 Turbo's limit is 4096 tokens per conversation. This limit dictates the AI's context, meaning once reached, the earliest parts of the conversation are omitted from the AI's consideration.

Promptplus will alert you to exlude previous messages when we detect you may go over the model's token API limit

In many AI interfaces, this isn't made explicit, potentially leading to confusion if the AI responses lack context from earlier messages. PromptPlus AI addresses this by keeping users informed about their token usage in a session, ensuring they are aware of the AI's current context.

Our interface allows users to manually select which prompts are included or excluded from the AI's context. This gives you control over the information the AI considers, optimizing your interaction within the token limit, and improving response quality. In short, understanding and managing token limits can significantly enhance your AI interaction.

Tokens management is a important concept to grasp for advance users, learn more by reading more on Tokens in this OpenAI Article

PARAMETER DESCRIPTION VALUE NOTES
temperature Governs the randomness of responses. Higher values create diverse but potentially less focused replies, while lower values yield more focused and predictable answers. 0 to 2 Can be thought of as creativity dial. Generally recommended altering this or top_p but not both.
top_p Controls the word range for the AI's response. Lower values make it stick to more common words, whereas higher values allow it to consider a broader vocabulary. 0 to 1 Generally recommended altering this or top_p but not both.
Frequency Penalty High values discourage the AI from repeating itself, whereas low values make the AI more likely to reiterate previous phrases or sentences -2.0 to 2.0 only apply when there is existing conversation history. They don't affect the AI's initial responses in a new conversation
Presence Penalty High values encourage the AI to bring up new topics in the conversation, while lower values make the AI stick to the existing topics. -2.0 to 2.0 only apply when there is existing conversation history. They don't affect the AI's initial responses in a new conversation

Note Don't worry about remembering the values on Promptplus AI, we have simply these values with a slider bar. For more information on these parameters, visit OpenAI's documentation.

Document Options link

You can access document options multiple ways: document creation, session, and within the dashboard for each individual session.

Document Options enables users to exercise a high degree of control over their sessions and has three tabs

Aside from the export tab, most of document options are availble to users initally when they create the session.

Export Tab

This tab focuses on export options for your sessions. You can choose to export the entire chat history or select specific parts to download, providing flexibility in managing and preserving your session data.

You can export your chat history in the following formats Text, HTML, JSON, CSV.

Note CVS and JSON can be uploaded to clone a session


Options Tab

This is where the majority of the session control tools reside

  • Edit title: Allows you to modify the title of your session.
  • Edit system message: Modify the system messages for a personalized experience. A system message gives context to a session without invoking a response.
  • upload: Use this feature to upload and integrate previous prompts into your current session. Make sure to use JSON or CSV.
  • Delete Session: Remove a session permanently.
  • Archive Session: If you want to preserve a session without any further prompt requests, use this option to make the session read-only.
  • Clone Session: Create an identical copy of the current session. This is especially useful when you want to try different approaches without modifying the original session.
  • Token Usage Stats: Gives you a detailed report on token usage for the session, helping you to monitor and manage your resource usage more effectively.
  • Clone Packet: Click to get a JSON of the packet info sent to API

Owner Tab

As an owner, you have additional tools at your disposal:

  • Add Labels: This feature allows you to classify your sessions using custom labels, which aids in the organization and filtering of your documents.
  • Owner's Note: A space for you to write notes related to the session. These can be reminders, observations, or any information pertinent to the session.

Profile Setting link

Overview

The Profile Options, accessible from both the chat interface and session dashboard, is an essential tool for personalizing and controlling your user experience in PromptPlus. The dialog presents the user's registered email at the top and the sign-out option at the bottom. It's structured into three tabs: User, Labels, and Account.


User

Use this tab to personalize your profile by changing your name and profile image

  • Name: Use the pencil icon to manually change your name, or the dice icon to generate a random one.
  • Profile Image: Upload a custom image or choose to generate a random mascot image.

In addition to these, there are four checkboxes for customizing display settings:

  • Large Text: Check this box to increase the size of the chat interface text.
  • Monospace Font: Switch the font family to monospace for chat prompts and responses.
  • Auto Exclude: This feature automatically excludes the oldest prompt in your chat session to avoid going over the API token limit for each ChatGPT model. This option is also available on a session basis inside document options > threhold

Labels

This section presents default labels (Personal, Business, and Archived) that you can assign to sessions during creation or afterwards. You can modify these labels according to your needs for better organization and filtering of your documents.


Account

This tab provides critical information about your usage and subscription.

Token Usage History: Provides a detailed breakdown of token usage by day, month, year, and all time based on sent messages and replies from the AI.

Monthly OpenAI Token Usage: Displays the total token limit and your usage for the month.

New Email: Allows you to migrate your document data to another email account.

Subscription Level: Shows your current subscription level. During the beta phase, all users are identified as Pre Release

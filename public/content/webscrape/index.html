Using ChatGPT for a How To on Web Scrape

One of the most exciting aspects of AI technology is its potential to help us learn and accomplish tasks more efficiently. Today, I'd like to share a real-world application of how an AI language model, specifically ChatGPT, helped me accomplish a task that would traditionally require a fair amount of technical knowledge and effort: web scraping.

Web scraping is a method used to extract large amounts of data from websites quickly. It can be a bit tricky if you're not familiar with programming languages like Python. However, with a bit of guidance from AI, even someone with a basic understanding of coding can accomplish this task.

Learning Python with AI Assistance

I started by asking ChatGPT to provide a Python code snippet that would allow me to scrape the required data.

The AI promptly provided me with a Python script using the requests and beautifulsoup4 libraries, along with an explanation of how the script works.

The next step was to run the script. I knew how to use an IDE (Integrated Development Environment) but wasn't sure how to execute the script. The AI guided me through the process, explaining how to install the necessary libraries and run the script in my IDE.

However, running the code didn't produce any results.

It turned out that the AI model had made an assumption about the HTML structure of the website that turned out to be incorrect.

I then provided the correct HTML elements that the script needed to select. With the corrected HTML elements, the AI model was able to provide a revised script.

But I wanted to take it a step further: I needed to categorize the contacts based on their titles and export the data to a CSV file for easier manipulation.

My message:

"I am using this list to create an email template. Detect if the title contains professor (variations like assistant or assiocaite are considered professors), if it does, give me another field called shortenTitle, and give them the title professor. if they have another title, we will refer to the original title field. I want to export this into a csv file."

The AI model was able to adjust the script to include these additional requirements.

Key Insights from AI-assisted Web Scraping

This experience demonstrated to me the power and flexibility of AI. The ability to ask questions and iteratively refine the solution enabled me to accomplish a task that might have been quite challenging on my own.

The takeaways from this interaction are:

  • Objective: Instruct the AI to scrape a site, clearly defining the goal.
  • Context: Provide enough context for the AI to perform the task. The website link and the correct HTML elements to look for were crucial in this case.
  • Desired output: Determine the desired output format, such as CSV for compatibility with Excel or Google Sheets.
  • Detailed requirements: Define any additional formatting rules, like shortening titles for specific purposes.

About the APP

Our application takes inspiration from our real-time game lobby example and incorporates these key features:

  • User Authentication: Secure and reliable user login system.
  • Session Creation: The ability to create new sessions with customizable options.
  • Session Joining: Seamlessly join existing sessions in the lobby.
  • User Profiles: Personalize your experience with adjustable settings
  • In-session Messaging: Facilitates interaction by enabling message exchanges with other participants.
  • Real-time User Status Updates: Keep track of user activity and engagement with presence indicators.

Typescript and Node

Leveraging the power of Typescript and Node, our application employs innovative NoSQL indexing techniques to manage real-time feeds and session listings efficiently. A deep path structure in the database allows for swift indexing and retrieval of session-related data.

Firebase

Hosted on Firebase, our app ensures data consistency and efficient retrieval. Firestore documents provide a scalable and efficient approach to store and retrieve data, ensuring a seamless experience for users. With its scale-on-demand hosting and robust data handling, choosing Google's Firebase was a logical choice for us. A NoSQL approach allows us to offer personalized user experiences, efficient updates, and seamless integration with session documents.

Accessibility

This is at the heart of our design philosophy. Our app ensures maximum accessibility with a password-free login and compatibility with any screen size or device - as long as it has a web browser. No applications to download, nothing to sign up for - just an email to register your account.
